# LINT — 위키 정합성 점검

> 프로젝트에 `CLAUDE.md`가 있고 LINT 워크플로를 정의했다면 **그 규칙을 우선**한다.

## 절차
1. `wiki/` 전반과 `index.md`를 점검해 다음을 찾는다:
   - 서로 **모순**되는 주장
   - **낡았거나** 더 이상 유효하지 않은 정보
   - **고아 페이지** (어디서도 `[[링크]]`되지 않음)
   - **누락된 상호링크** (본문에서 언급됐는데 링크 안 된 페이지)
   - **지식 공백** (stub 상태로 방치된 페이지)
2. 결과를 한국어로 보고하고, 명확한 항목은 즉시 수정한다(수정 시 `updated` 갱신).
3. `log.md`에 LINT 줄을 **추가만** 한다:
   ```
   YYYY-MM-DD LINT 모순 N, 고아 N, 누락 링크 N
   ```
